7 class Sample():
8  '''Measurement data read by the implant at one point in time,
9  i.e. for one sample.
10 
11  An instance holds the following information:
12 
13  - Number of channels used for measuring. Array measurements
14  must be of this size.
15  - Array of measurements. This array must be of size
16  numberOfMeasurements.
17  - Supply voltage in mV.
18  - Information whether the implant is connected
19  (all channels available).
20  - Id of the stimulation which starts with this sample. If no
21  stimulation started with this sample, then the
22  stimulationNumber is IMPLANT_NO_STIMULATION.
23  - Information whether the stimulation is active.
24  - Counter that is increased for each measurement sample starting with 0.

97  '''Class holding callback functions which are called by the implant
98  to notify the user on certain events.
99 
100  For example, new measurement data, implant state changes or
101  implant health state changes from an implant. It is possible to
102  initialize single handles as NULL before creating the listener
103  handle in order to not receive any events of a certain type. If
104  the callbacks are not NULL they must point to callable
