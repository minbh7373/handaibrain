Detailed Description

Iterator interface to iterate through all atoms of a stimulation function.

A stimulation atom iterator can be generated via stimulationfunction_getAtomIterator (

See also
stimulationfunction.h). The iterator must be destroyed via stimulationatomiterator_destroy once it is no longer needed.
