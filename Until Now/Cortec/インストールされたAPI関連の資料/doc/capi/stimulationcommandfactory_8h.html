Detailed Description

Factory interface for creation of stimulation commands, functions and atoms.

Typical usage:

  1. Create empty stimulation command.
  2. Create empty stimulation function. Set repetition parameter.
  3. Create stimulation atoms and append them to function.
  4. Append stimulation function to command.
  5. Repeat steps 2. - 4. until all functions are created.

Creates a stimulation atom. Each atom represents a constant amplitude for a fixed time duration.

Note: 1. The caller must take care for destruction of the stimulation atom once it is not longer needed. E.g. via stimulationatom_destroy (

See also
stimulationatom.h) or by passing the function to a stimulation function, which takes care of the destruction of its atoms.
  1. Atoms of different types cannot be appended to the same stimulation function

Example: Using createRectStimulationAtom(12, 2000000) would return a stimulation atom that stimulates with 12 micro ampere for 2 seconds.
