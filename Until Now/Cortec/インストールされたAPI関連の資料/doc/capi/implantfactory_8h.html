Detailed Description

Device discovery and factory interface creating implant handles to control implant devices.

Main responsiblities of this component:

  • discovering all external units and implants connected to the system which are recognized by the factory
  • creating an instance of a suitable hardware abstraction layer implementation for a specified external unit and implant combination.
