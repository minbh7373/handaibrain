Detailed Description

Device discovery and factory interface creating instances of IImplant.

As braincon is intended as platform technology, there will be several different hardware abstraction layers implementations. The implementations of this factory interface are capable to

  • discover all external units and implants connected to the system which is recognized by the factory
  • create an instance of a suitable hardware abstraction layer implementation for a specified external unit and implant combination.
