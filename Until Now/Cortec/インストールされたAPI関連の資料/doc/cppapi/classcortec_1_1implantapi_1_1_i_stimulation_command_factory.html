Detailed Description

Factory interface for creation of stimulation commands.

An instance of IStimulationCommandFactory can be used to compose stimulation commands and stimulation functions.

Typical usage:

  1. Create empty stimulation command.
  2. Create empty stimulation function. Set repetition parameter.
  3. Create stimulation atoms and append them to function.
  4. Append stimulation function to command.
  5. Repeat steps 2. - 4. until all functions are created.

Creates a "Rectangular" stimulation atom. Each atom represents a constant amplitude for a fixed time duration.

Consider an implant that does voltage stimulation. Then createRectStimulationAtom(12, 2000000) would return a stimulation atom that stimulates for 2 seconds with an amplitude of 12 Volts. However, if this atom would be sent to an implant that does current stimulation, this atom would elicit a stimulation with 12 micro ampere for 2 seconds.

Parameters
[in]amplitudeStimulation amplitude, either in V or uA.
[in]durationDuration in micros of the stimulation.
