[in]numberOfMeasurementsNumber of channels used for measuring. Array measurements must be of this size.
[in]measurementsArray of measurements. This array must be of size numberOfMeasurements. Note: The ownership of measurements is passed to CSample
[in]supplyVoltageMilliVSupply voltage in mV.
[in]isConnectedTrue if the implant is connected (all channels available)
[in]stimulationNumberid of the stimulation which starts with this sample. If no stimulation started with this sample, then the stimulationNumber is Sample::c_noStimuation.
[in]stimulationActiveTrue if the stimulation is started.
[in]counterCounter that is increased for each measurement sample starting with 0. The value range is 0, 4294967295. If the maximum value is exceeded the counter will be reset automatically.
